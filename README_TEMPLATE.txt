Portfolio Comic-Style Template Template portfolio modern dengan gaya
komik, border tebal, halftone background, dan animasi interaktif.
Dibangun menggunakan Next.js, React, dan Tailwind CSS.

Fitur Utama - Halftone background ala Komik - Hero section
bergaya komik - Navbar interaktif dengan active highlight - Section
About, Project, dan Contact - Footer dengan comic block shadow - Fully
responsive - AOS animation support - Clean folder structure - Siap
dipakai dan mudah dikustomisasi

Teknologi yang Digunakan - Next.js 14 - React 18 - Tailwind CSS - AOS
Animation - React Icons

Instalasi 1. Clone atau download project git clone
https://your-repo-link.com cd your-folder

2.  Install dependencies npm install

3.  Jalankan development server npm run dev

4.  Buka di browser http://localhost:3000

Struktur Folder / ├── app/ │ ├── page.tsx │ ├── globals.css │ ├──

Cara Edit Konten Edit nama dan teks hero Buka: app/components/Hero.tsx

Ubah: Welcome to My Portfolio Discover my work and learn more about me.

Edit Navbar Buka: Navbar.tsx Ubah nama: Your Name

Edit About Section Buka: About.tsx

Edit Projects Buka: Projects.tsx Tambah atau ubah project card dari
array list.

Edit Contact Buka: Contact.tsx Anda dapat mengganti email, link WA, atau
formulir.

Edit Footer Buka: Footer.tsx

Mengganti Gambar Simpan file gambar di folder: public/

Lalu panggil:

Build untuk Production npm run build npm run start

Deployment Template ini bisa dideploy di: - Vercel (recommended) -
Netlify - Host static Next.js lainnya

Deploy via Vercel: npm i -g vercel vercel

Lisensi Template ini dilisensikan untuk penggunaan pribadi atau
komersial oleh pembeli sah. Tidak diperbolehkan mengklaim sebagai
pembuat asli tanpa izin.
